# Contextual Clarity AI | Prank Edition

This website is a "Malapropism Prank Tool" disguised as a high-end Silicon Valley writing assistant. It is designed to subtly sabotage a user's writing by swapping correctly used words for their most common "confusables" (homophones).

## 🚀 How it Works
1. **The Professional Mask:** The UI is clean, modern, and mimics high-tier productivity software (like Notion or Grammarly).
2. **Semantic Sabotage:** The JavaScript engine uses a "Reverse Confusion Map." 
   - If you use **"there"** correctly, it swaps it for **"their"**.
   - If you use **"accept"**, it swaps it for **"except"**.
   - If you use **"lose"**, it swaps it for **"loose"**.
3. **The Result:** The text remains readable and "correct-looking" to a spell-checker, but the meaning becomes entirely wrong or embarrassing.

The logic relies on a Regex-based tokenizer that separates words from punctuation. It then performs a case-sensitive lookup against a dictionary of 24+ common linguistic traps.
